# AI Data Analysis Assistant

## What it does:
Upload any Excel or CSV file. AI analyzes it and gives you smart insights.

## How to use:
1. Enter Gemini API key
2. Upload your file
3. Click analyze

## Skills shown:
- Python
- Google Gemini AI
- Data Analysis (Pandas)
- Streamlit Web App
